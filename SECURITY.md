# Security Policy

## Reporting a Vulnerability

We take the security of `gologaggregator` seriously. If you discover a security vulnerability, please send an email to **enoque.sousa@gmail.com** (or open a private issue if GitHub supports it).

Please include:
- A detailed description of the vulnerability.
- Steps to reproduce (Proof of Concept).
- The potential impact.

### Our Commitment

- We will respond to your report within 48 business hours.
- We will keep you informed about the progress of the fix.
- We will not take legal action against researchers who act in good faith and follow this policy (responsible disclosure).

## Supported Versions

Only the latest version on the `main` branch receives security updates.
